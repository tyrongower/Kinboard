### Security Policy

If you believe you have found a security vulnerability in Kinboard, please email:

- security contact: security@your-domain.example (update this to your project email)

Please include a description of the issue, steps to reproduce, and any relevant logs or PoC. We will acknowledge receipt within 5 business days and work with you on a fix and coordinated disclosure.

Please do not open public GitHub issues for sensitive security reports.
